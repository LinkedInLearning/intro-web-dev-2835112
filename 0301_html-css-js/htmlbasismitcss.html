Sprachen im Web

HTML

HTML steht für Hypertext Markup Language und ist die Basis aller Webseiten. Es ist eine Auszeichnungssprache (to markup = auszeichnen). HTML dient dazu, die Bedeutung der einzelnen Texte zu benennen - also: das hier ist eine Überschrift, das ist ein Absatz, das hier ist eine Liste und hier sind Links. Außerdem gibt es Möglichkeiten, die groben Bereiche zu definieren (Kopfbereich, Navigation) und besondere Auszeichnungen fügen Bilder ein oder erstellen Formulare.

CSS

HTML bestimmt die Funktion von Texten und gibt die grundlegende Struktur eine Webseite vor. Damit eine Website aber attraktiv aussieht, braucht es eine weitere Sprache: und das ist CSS. Die Abkürzung "CSS" steht für Cascading Stylesheets, es ist die Formatierungssprache im Web. Mit CSS können Sie die Farben von Bereichen definieren, die Schriftart und Bereiche anordnen; aber es sind auch fortgeschrittene Techniken möglich wie beispielsweise ausgefeilte Animationen.

JavaScript

JavaScript ist eine Programmiersprache und ist die dritte wichtige Sprache im Web. Mit JavaScript können Sie:

  • ausgefeilte Überprüfungen von Formularen durchführen
  • Browser-Spiele erstellen
  • Nutzereinstellungen speichern und an einen Server schicken
  • Daten von Servern laden

Mit JavaScript können Sie zum Beispiel definieren, dass etwas geschehen soll, wenn jemand auf einen Button klickt.
